## Motivation

`NestJS`를 좀 더 정확히 이해하기 위해선 **앵귤러의 모듈 구조**를 알면 좋다고 한다. 또한 NestJS 공식 홈페이지에서도 `The architecture is heavily inspired by Angular` 라고 적혀있었다는 점은 내가 한번쯤 앵귤러를 맛보기에 충분한 필요조건이 된다고 생각한다.
